**这里主要采用双指针方法，来不断刷新数组（覆盖旧数据）。**

双指针主要用法：

- **双头指针** 
  ```python
  fast, slow = 0, 0
  ```
- **首尾指针**
  ```python
  left, right = 0, len(nums)-1
  ```

```python
# 27 移除元素
# 示例 1: 给定 nums = [3,2,2,3], val = 3, 
# 函数应该返回新的长度 2, 并且 nums 中的前两个元素均为 2。 
# 你不需要考虑数组中超出新长度后面的元素。
# 示例 2: 给定 nums = [0,1,2,2,3,0,4,2], val = 2, 
# 函数应该返回新的长度 5, 并且 nums 中的前五个元素为 0, 1, 3, 0, 4。
示例 1: 给定 nums = [3,2,2,3], val = 3, 函数应该返回新的长度 2, 并且 nums 中的前两个元素均为 2。 你不需要考虑数组中超出新长度后面的元素。

示例 2: 给定 nums = [0,1,2,2,3,0,4,2], val = 2, 函数应该返回新的长度 5, 并且 nums 中的前五个元素为 0, 1, 3, 0, 4。
def removeElement(nums,val):
    n = len(nums)
    fast, slow = 0, 0
    while fast < n:
        if nums[fast] != val:
            # 若不是待删除元素，快指针给慢指针赋值
            nums[slow] = nums[fast]
            slow += 1
            #若是删除元素，慢指针不动，快指针继续寻找下一个非删除元素赋值过来
        fast += 1
    return slow
```

```python
# 26.删除排序数组中的重复项
# 示例 1:
# 给定数组 nums = [1,1,2],
# 函数应该返回新的长度 2, 并且原数组 nums 的前两个元素被修改为 1, 2。
# 你不需要考虑数组中超出新长度后面的元素。
# 示例 2:
# 给定 nums = [0,0,1,1,1,2,2,3,3,4],
# 函数应该返回新的长度 5, 并且原数组 nums 的前五个元素被修改为 0, 1, 2, 3, 4。
# 你不需要考虑数组中超出新长度后面的元素。
def removeDuplicates(nums):
    n = len(nums)
    fast, slow = 1, 0
    while fast < n :
        if nums[fast] == nums[slow]:
            # 相等的时候，快指针加一
            fast += 1
        else:
            # 不相等时，先慢指针加一，快给慢指针赋值，快指针加一
            slow += 1
            nums[slow] = nums[fast]
            fast += 1
            # 这里slow是最后一个不重复元素的下标，要求返回的是长度。
            # 所以这里返回要加一
    return slow+1
```

```python
# 283 移动零
# 示例:
# 输入: [0,1,0,3,12]
# 输出: [1,3,12,0,0] 
def moveZeroes(nums):
    n = len(nums)
    fast, slow = 0, 0
    # 第一遍刷双指针把非0元素都移动到靠前位置。
    for fast in range(n):
        if nums[fast] == 0:
            fast += 1
        else:
            slow += 1
    # 第二遍历，把slow 后面的元素全部置零，也就是把非0元素以外的元素置零。
    for i in range(slow,n):
        nums[i] = 0
    return nums
```

```python
#  844 - 比较含退格的字符串
# 示例 1
# 输入：S = "ab#c", T = "ad#c"
# 输出：true
# 解释：S 和 T 都会变成 “ac”。
#  示例 2
# 输入：S = "ab##", T = "c#d#"
# 输出：true
# 解释：S 和 T 都会变成 “”。
def backspaceCompare(nums1,nums2):
    def judge(nums):
        n = len(nums)
        slow, fast = 0, 0
        for fast in range(n):
            if nums[fast] != "#":
                # 如果快指针遇到非“#”，不需要回退
                # 把当前快指针元素替换给慢指针对应元素
                # 这里注意只replace一个元素！
                nums = nums.replace(nums[slow],nums[fast],1)
                slow += 1
            else:
                # 如果快指针遇到“#”，回退
                slow -= 1
                # 这里判断slow < 0, 返回空字符串
        return nums[:slow] if slow > 0 else ""
    return judge(nums1) == judge(nums2)
```

```python
# 977.有序数组的平方
# 输入：[-4,-1,0,3,10] 输出：[0,1,9,16,100]
# 输入：[-7,-3,2,3,11] 输出：[4,9,9,49,121]
def sortedSquares(nums):
    # 定义收尾的双指针
    left, right = 0, len(nums)-1
    while left < right:
        # 对比收尾元素的平方
        if abs(nums[left] ** 2) >= abs(nums[right] ** 2):
            # 若首指针的平方大于尾指针的平方，
            # 1、先置换首尾元素
            # 2、将平方之后元素赋值给尾指针
            # 3、继续尾指针-1
            # 这里没有首指针+1，而进行尾指针-1，是因为step2已经处理过尾指针元素
            # 必须是移动尾指针。
            # ！！！ 这指针始终都是指向待处理元素
            nums[right], nums[left] = nums[left], nums[right]
            nums[right] = abs(nums[right] ** 2)
            # left += 1
            right -= 1
        else:
            nums[right] = abs(nums[right] ** 2)
            right -= 1
    # print(nums)
    return nums
```
